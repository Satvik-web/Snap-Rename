How to use the App On any Platform?

Terminal Commands:
-> Use srename command in terminal.

-> srename -d [Working Directory]
    -> Ex: srename -d ~/Downloads
This opens a tui application with your specified directory
Without the -d flag, it will ask you to specify the directory required.

-> srename --gui
    -> Opens it in Gui.
    -> You can use -d [Working Directory flag to open you specific directory in gui]
    -> Without -d flag, it will ask you to browse a folder.

Directly Opening the App:
-> You can open the app from the Start Menu of Windows, Linux and from Applications on mac.
-> It will ask you to select a folder to open.

Using Windows File Explorer:
-> Select a folder -> Right Click -> Show Other Options (For Windows 11) -> Snap Rename 
This opens that folder in the app. 

Using Mac Finder:
-> Select a Folder -> Right Click -> Quick Actions -> Snap Rename 
This opens that folder in the app.
