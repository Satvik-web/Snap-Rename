Basic Info:
linux_installer.sh -> Installs the app on any kind of linux
srename.sh -> Terminal communicate utility.

linux_app.zip Contains: 1. All the files in Python Version 2. srename.sh

The installer actually downloads linux_app.zip from GitHub Releases and installs it.

How to install?
1. Download linux_installer.sh from GitHub Releases.
2. Open Terminal and navigate to the folder where you downloaded the file.
3. Run chmod +x linux_installer.sh
4. Run ./linux_installer.sh
5. Follow the instructions on the screen.
