Basic Info:
Application Folder -> Contains: 1. The code for the Application under Projects 2. windows_app.zip (The compiled application)

Installer Folder -> Contains 1. The code for the installer under Projects 2. windows_installer.exe (The compiled installer)

Important Notes:

1. Snap Rename -> The app that launches main.py in gui mode.
2. srename -> Terminal based app that launches main.py in terminal

3. The apps always launch python in admin mode so that the user can rename files in any folder without much clashes.

4. windows_app.zip -> Contains: 1. The compiled application, both srename and Snap Rename in the same folder. 2. All the python files found in the Python Version folder. 3. Link File to Snap Rename.exe (to easily place it anywhere) 4. logo.ico and logo.png 

5. The installer actually downloads windows_app.zip and installs it.

How To Install? 
1. Download windows_installer.exe from GitHub Releases.
2. Double Click on it.
3. Follow the instructions on the screen.
