Snap Rename — Overview

Snap Rename is a powerful browser-based batch file renaming tool. It lets you rename dozens of files at once using intelligent rules — from simple find & replace to complex metadata-driven templates — all with a live preview of every change before it's applied.

⚡

See every rename change in real time as you configure it.

Getting Started

Loading a Workspace

Snap Rename works with a workspace folder — a directory on your computer containing the files you want to rename. The app uses the browser's File System Access API to read and write files directly on your disk (Chromium browsers only).

1
Click "Directory…" in the top toolbar of the file pane. A system folder picker will appear.
2
Select your folder and grant permission when the browser asks.
3
All files in that folder load into the file table instantly. Hidden files (starting with .) are filtered out automatically.
Tip: Snap Rename only reads the top-level folder — it does not recurse into subfolders.

Getting Started

File Table

The file table shows all files in your workspace. Each row shows the file's current (or previewed new) name, date modified, size, and file type.

Sorting

Use the Sort by dropdown in the toolbar to sort files by:

Alphabetical Date Added Date Modified Size Extension

Selection

  • Single click — selects that file (deselects others).
  • Ctrl/⌘ + click — adds/removes a file from the selection.
  • Double-click — opens the file in the default system application.

Colour coding

When a rename operation is in the pipeline, filenames that will change are shown in red. Files that won't be affected remain white.

Getting Started

Preview Pane

The Preview Pane (third column) shows information about the currently selected file. When multiple files are selected, it shows data for the topmost selected file.

  • File icon — emoji based on file type (🖼️ Image, 🎵 Audio, 🎬 Video, 📄 Document, etc.)
  • File name — current name of the file.
  • Kind & Size — file type and size in human-readable units.
  • Created / Modified — timestamps from the file system.

Click "Hide Preview" in the toolbar to collapse the pane and gain more space for the file table.

Getting Started

Pipeline

The Pipeline lets you stack multiple rename operations and run them in sequence on every file. Operations are applied left-to-right (top-to-bottom in the sidebar list).

1
Configure any tool in the right pane.
2
Click "+ Add to Pipeline". The step appears in the Active Pipeline list in the sidebar.
3
Repeat for as many tools as needed.
4
The file table and live previews update instantly to show cumulative results.
5
To remove a step: click it in the pipeline list to select it (highlighted), then click "Remove Step". If nothing is selected, the last step is removed.
Example pipeline: Remove Extra Spaces → Capitalize First Letters → Add Prefix "2024_" → Add Suffix " (Final)" — all applied in one click.

Ease of Usage

Beyond opening the app from your Start Menu or Applications folder, Snap Rename integrates directly into your system's file explorer for instant access.

1
Windows File Explorer: Select a folder, right-click, and select Show Other Options (Windows 11). Look for Snap Rename in the menu to open that directory instantly.
2
macOS Finder: Select a folder, right-click, and navigate to Quick Actions > Snap Rename. The app will launch pre-loaded with your chosen files.
3
Direct Launch: You can always launch the app via your system search (Spotlight, Start Menu, etc.) and use the Directory... button to browse for a folder.
Application

Terminal Commands (CLI)

For power users and developers, the srename command is provided system-wide after installation. This allows you to launch the renamer directly from your terminal or include it in scripts.

-d [path]
Specifies the Working Directory to load on launch.
srename -d ~/Downloads
--gui
Forces the Desktop GUI to open. Without this flag, the TUI is preferred.
srename --gui
CLI Troubleshooting:
  • "command not found: srename": Ensure your PATH includes the installation directory (usually handled by the installer).
  • Python Errors: If running from source, ensure PyQt6 and textual are installed in your active environment.
  • Permission Denied: On Linux/macOS, you may need to grant execution permissions: chmod +x srename.

Terminal Interface (TUI)

Snap Rename features a high-performance Terminal User Interface (TUI) built with the Textual framework, mirroring the full functionality of the GUI. It's designed for keyboard-centric workflows, offering lightning-fast navigation without the need for a mouse.

TUI Interface Overview

Keyboard Shortcuts

Mastering these shortcuts allows for lightning-fast batch renaming without touching a mouse:

KeyAction
dDirectory: Open a modal to change the working folder
p+Pipeline: Add the current tool settings to the active pipeline
aApply: Run the entire pipeline on every file in the directory
uUndo: Revert the last batch rename operation
cClear: Remove all steps from the current pipeline
rRemove Step: Delete the last added operation from the pipeline
1 - 6Switch Tools: Quickly jump between Clean, Smart, Normal, etc.
qQuit: Exit the application safely
Live Preview: All TUI operations generate the same live previews as the GUI, highlighting pending changes in bold red before generating files.

The TUI is meticulously optimized for cross-platform terminal compatibility, including support for Windows CMD, PowerShell, and various Linux/macOS shells.

Command Palette & Shortcuts

Snap Rename includes built-in "Spotlight" style searching to help you navigate features and documentation quickly.

TUI Command Palette

In the Terminal UI, you can trigger the Command Palette by pressing Ctrl + P (or Cmd + P on macOS) or by clicking the small circle icon in the top-left corner.

  • Search Actions: Type the name of any tool (e.g., "Smart", "Clean") or action (e.g., "Apply", "Undo") to jump to it instantly.
  • No Shortcuts Needed: If you forget a keyboard binding, the palette lists all available commands in a searchable list, making the app accessible even without memorizing keys.

Rename Tools

Enhanced Clean Filename

A one-stop cleanup tool that removes unwanted content and normalises filenames. Options can be combined.

Remove Extra Spaces
Collapses multiple consecutive spaces, underscores, or dashes into one. Trims leading/trailing spaces.
My__File Name.jpg → My_File Name.jpg
Remove Duplicate Words
Removes repeated words (case-insensitive) that appear consecutively in the filename stem.
photo photo summer.jpg → photo summer.jpg
Remove Special Characters
Strips all characters that aren't alphanumeric, spaces, underscores, or hyphens.
IMG@2024!.jpg → IMG2024.jpg
Remove All Numbers
Deletes every digit from the filename stem.
Track01.mp3 → Track.mp3
Remove All Letters
Deletes every alphabetical character from the filename stem.
Track01.mp3 → 01.mp3
Normalize Format → Spaces
Replaces all underscores and dashes with spaces.
my_file-name.txt → my file name.txt
Normalize Format → Dashes
Replaces all spaces and underscores with dashes.
my file_name.txt → my-file-name.txt
Normalize Format → Underscores
Replaces all spaces and dashes with underscores.
my file-name.txt → my_file_name.txt
Casing → Capitalize First Letters
Title-cases every word in the filename stem.
my summer photo.jpg → My Summer Photo.jpg
Casing → Uppercase
Converts the entire filename stem to uppercase.
invoice.pdf → INVOICE.pdf
Casing → Lowercase
Converts the entire filename stem to lowercase.
INVOICE_FINAL.pdf → invoice_final.pdf

Rename Tools

Smart Find & Replace

The most powerful tool in Snap Rename. Select a Find Pattern (what to target) and an Action (what to do with it). The pattern options cover 21 built-in detectors as well as custom regex.

The live preview at the bottom of the tool pane always shows the result for the currently selected file (or the first file if none is selected).
Smart Find & Replace Overview
Smart Pattern

1. Numbers

Targets every sequence of digits (\d+) in the filename stem.

Track01_Summer2024.mp3 → Track_Summer.mp3 Remove
IMG_001.jpg → IMG_NUM.jpg Replace With "NUM"
Smart Pattern

2. Letters

Targets every sequence of alphabetical characters ([a-zA-Z]+).

Track01.mp3 → 01.mp3 Remove
Smart Pattern

3. Spaces

Targets every single space character.

My Summer Photo.jpg → My_Summer_Photo.jpg Replace With "_"
Smart Pattern

4. Special Characters

Targets any character that is not alphanumeric, a space, an underscore, or a dash ([^\w\s-]).

IMG@2024!.jpg → IMG2024.jpg Remove
Smart Pattern

5. Dates

Targets common date patterns: YYYYMMDD, YYYY-MM-DD, YYYY_MM_DD.

photo_2024-07-15_holiday.jpg → photo_holiday.jpg Remove
Smart Pattern

6. Brackets / Parentheses

Targets content inside [], (), {}, or <> — including the brackets themselves.

Movie (2023) [BluRay].mkv → Movie .mkv Remove
Smart Pattern

7. Consecutive Spaces

Targets runs of 2 or more spaces ( {2,}), leaving single spaces intact.

My File.txt → My File.txt Replace With " "
Smart Pattern

8. Underscores / Dashes

Targets one or more consecutive underscores or dashes ([_-]+).

my_file--name.txt → my file name.txt Replace With " "
Smart Pattern

9. File Extension

Targets the file extension (e.g., .mp3). Use Remove to strip it, or Replace With to change it to a new extension.

audio.mp3 → audio.flac Replace With ".flac"
Changing extensions doesn't convert file formats — it simply renames the extension string.
Smart Pattern

10. Non-ASCII Characters

Targets any character outside the standard ASCII range (code points above 127) — e.g., accented letters, Chinese characters, emoji.

résumé_📄.pdf → rsum_.pdf Remove
Smart Pattern

11. Leading / Trailing Spaces

Targets spaces at the very start or end of the filename stem. Use Remove to strip them.

Invoice .pdf → Invoice.pdf Remove
Smart Pattern

12. Leading Numbers

Targets digits (and optional separator) at the very start of the filename stem (^\d+[\s_-]*).

01_Track Name.mp3 → Track Name.mp3 Remove
Smart Pattern

13. Trailing Numbers

Targets digits (and optional separator) at the very end of the filename stem ([\s_-]*\d+$).

Photo 02.jpg → Photo.jpg Remove
Smart Pattern

14. Leading / Trailing Underscores

Targets underscores at the very beginning or end of the filename stem (^_+|_+$).

_filename__.txt → filename.txt Remove
Smart Pattern

15. Capitalize First Letter

Capitalizes only the very first character of the filename stem, leaving the rest unchanged.

my document.pdf → My document.pdf Standardize
Smart Pattern

16. Uppercase All Letters

Converts every letter in the filename stem to uppercase.

invoice_final.pdf → INVOICE_FINAL.pdf Standardize
Smart Pattern

17. Lowercase All Letters

Converts every letter in the filename stem to lowercase.

INVOICE_FINAL.pdf → invoice_final.pdf Standardize
Smart Pattern

18. Swap Words

Swaps the first and second word of the filename stem (separated by space, underscore, or dash).

Smith_John.pdf → John_Smith.pdf Standardize
Smart Pattern

19. Character Position

Inserts text at a specific 1-based character position within the filename. When adding to the pipeline, a dialog asks whether the position counts characters in the stem only or the full filename (including extension).

If the position you enter is greater than the length of the target string, a warning modal appears and the operation is not added.
Photo.png → Phohito.png Insert "hi" at position 3 (stem)
Photo.png → Ph_oto.png Insert "_" at position 2 (stem)
Character Position Insert
Smart Pattern

20. Custom Exact

Enter any literal text string to match. All occurrences in the filename stem are targeted. No special regex characters are interpreted.

IMG_FINAL_v2_FINAL.jpg → IMG__v2_.jpg Remove "FINAL"
Smart Pattern

21. Custom Regex

Write any valid JavaScript regular expression. The pattern is applied globally to the filename stem. If the regex is invalid, the operation is skipped gracefully.

Track-001-Summer.mp3 → Track_001_Summer.mp3 Regex - → Replace With "_"
photo_v2024.jpg → photo_.jpg Regex v\d{4} → Remove
Rename Tools

Smart Find & Replace — Actions

Once you've chosen a Find Pattern, select what to do with it:

Remove
Deletes every match from the filename stem.
Track 01 Mix.mp3 → Track Mix.mp3 (Remove Numbers)
Replace With
Substitutes every match with the text you enter in the replacement field.
my file.txt → my_file.txt (Spaces → Replace With "_")
Insert Before
Inserts the replacement text immediately before each match.
001.mp3 → [001].mp3 (Numbers → Insert Before "[")
Insert After
Inserts the replacement text immediately after each match.
001.mp3 → 001].mp3 (Numbers → Insert After "]")
Standardize
Used with casing patterns (Capitalize First Letter, Uppercase, Lowercase, Swap Words). Applies the casing transformation.
hello world.txt → Hello World.txt
Extract
Keeps only the matched portions, discarding everything else from the stem.
abc123def456.txt → 123456.txt (Extract Numbers)
Insert (Stem Only)
Used with Character Position. Inserts text at position N counting only within the filename stem (not the extension).
Photo.png → Phohito.png (pos 3, insert "hi")
Insert (with Extension)
Used with Character Position. Inserts text at position N counting from the very start of the full filename including the extension.
Photo.png → Pho[to.png (pos 3, insert "[")
Actions Demo
Rename Tools

Normal Find & Replace

A simple, literal text replacement. No patterns or regex — just type what you want to find and what to replace it with.

Find
The exact text to search for in the filename stem. All occurrences are replaced.
Replace With
What to substitute each match with. Leave blank to delete the matched text.
Case Sensitive
When checked, "Card" and "card" are treated as different. When unchecked, both match.
Aadhar Card.pdf → Aadhar TEST.pdf Find "Card" → Replace "TEST"
Normal Find & Replace
Rename Tools

Prefix / Suffix

Add constant text to the beginning (prefix) and/or end (suffix) of every filename stem. The extension is never affected.

Prefix Add
Text inserted before the filename stem.
Prefix "2024_" → Photo.jpg → 2024_Photo.jpg
Suffix Add
Text appended after the filename stem (before the extension).
Suffix " (Final)" → Report.pdf → Report (Final).pdf
Both fields can be filled simultaneously to add a prefix and suffix in one step.
Prefix / Suffix
Rename Tools

Sequential Numbering

Automatically append or prepend an incrementing number to each file, perfect for ordering photos, tracks, or scans.

Placement
At Front — number comes before the name. At End — number comes after the name.
At Front: 001_Track.mp3 / At End: Track_001.mp3
Base Name
If filled, replaces the original filename stem for all files. Leave blank to keep each file's original name.
Base "Photo" → Photo_001.jpg, Photo_002.jpg…
Start
The number to begin counting from. Default is 1.
Pad
Minimum digit width, zero-padded. Pad 3 = 001, 002… Pad 1 = 1, 2…
Sequential Numbering

Apply & Undo

Applying Renames

When you're satisfied with the preview, click "Apply Active Renames".

1
If some files are selected: a dialog asks whether to rename only those files or all files.
2
If no selection: a confirmation modal asks you to confirm renaming all files in the workspace.
3
Each rename creates a new file with the new name containing the original file's data, then deletes the old file. This ensures compatibility with the browser's File System Access API.
4
A success modal reports how many files were renamed and how many (if any) failed.
5
The pipeline is automatically cleared after a successful rename.

Conflict Resolution

If two files would end up with the same name, the second one is automatically renamed with a numeric suffix (e.g., file(1).txt, file(2).txt) so no data is ever lost.

Applying Renames
Apply & Undo

Undo

Click "Undo Last Rename" at any time after applying a batch to revert all renamed files back to their original names.

  • Undo is tracked per session — it only works while the page is open.
  • Only one level of undo is supported (the most recent batch).
  • Undo works the same way as rename: creates the old-named file, copies content, deletes the new-named file.
Refreshing or closing the page clears the undo history. Always verify your rename preview before applying.
Apply & Undo

Drag & Drop

You can drag files from Finder (or any file manager) directly into the file table area to copy them into the current workspace.

  • If a file with the same name already exists you'll be asked whether to replace it or keep both (the copy is renamed with a suffix like filename copy 1.ext).
  • Dropped files are immediately added to the workspace and show up in the table.
You must have a workspace folder selected before dragging files in.
